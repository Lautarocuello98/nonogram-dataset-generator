# Nonogram Dataset Generator (Demo)

A Python tool for generating **Nonogram (Picross) puzzle datasets** with PNG rendering and structured JSON metadata.

This repository is a **public demo version** that showcases the dataset structure and example outputs produced by the generator.

The full generation engine and extended dataset pipeline are kept in a private repository.

---

## Overview

This project generates Nonogram puzzles programmatically and exports them in two formats:

• **PNG images** showing the puzzle grid and clues  
• **JSON metadata** containing the full puzzle representation  

The generated dataset can be used for:

- puzzle platforms
- algorithm testing
- dataset creation
- machine learning experiments
- puzzle solving research

## Dataset Structure

Each puzzle includes a JSON file with structured metadata.

Example:

```json
{
  "id": "5x5_0001",
  "size": 5,
  "grid": [
    "01110",
    "11111",
    "01110",
    "01110",
    "00000"
  ],
  "grid_encoding": "binary_rows",
  "row_clues": ["3", "5", "3", "3", "0"],
  "col_clues": ["1", "4", "4", "4", "1"],
  "filled_cells": 14,
  "empty_cells": 11,
  "density": 0.56,
  "image_file": "puzzle_0001.png"
}
```

Each puzzle directory also contains the corresponding PNG image.

## Example Output Structure

```
output/
   5x5/
      puzzle_0001.png
      puzzle_0001.json

```

The `manifest.json` file contains dataset metadata including:

- generation timestamp
- generation seed
- number of puzzles generated per size
- total puzzle count

## Usage

Run the generator from the command line:

```bash
python main.py --sizes 5 10 15 --count-per-size 10
```

### Parameters

| Argument | Description |
|--------|--------|
| `--sizes` | Puzzle sizes to generate |
| `--count-per-size` | Number of puzzles per size |
| `--seed` | Random seed for reproducibility |

Example:

```bash
python main.py --sizes 5 10 --count-per-size 5
```

- Procedural puzzle generation
- Grid validation filters
- Density constraints

- PNG rendering with row/column clues
- JSON dataset export
- Reproducible generation via random seed
