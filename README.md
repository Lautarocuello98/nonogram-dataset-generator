# Nonogram Dataset Generator (Demo)

This repository is in public demo mode.

## What is included

- Project overview
- Dependencies
- One real output sample (`output/5x5/puzzle_0001.*`)
- `manifest.json` to show the dataset structure

## What is not included

The full source code and test suite are kept in a private version.

## Stack

- Python
- NumPy
- Pillow
- tqdm

## Project goal

Generate Nonogram (Picross) datasets in PNG + JSON format, including row/column clues and metadata for analysis.
